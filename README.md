# Profit-decline-diagnostic - Business Performance Analysis 

## Problem Statement 
Sales performance is strong, but overall profitability is inconsistent. This project aims to identify the key drivers of profit decline and data driven decision making.

## Tools used 
- Excel 
- Power BI 

## Analysis Overview 
- Cleaned and prepared sales data using Excel
- Analyzed profitabilty across regions, product categories, and customers segments
- Examined the impact of discounting on profit margins
- Built executive level Power BI dashboard for business insights

## Key Insights 
- High discounting strongly correlates with negative profit margins
- Certain sub categories contribute disproportionately to overall losses
- A small number of products drive most of the negative profit

## Outcome 
The dashboard enables management to identify loss making areas and take corrective actions.
